展示追番列表的 WordPress 插件，使用短代码 [bangumi] 即可在文章或页面上展示自己在 Bangumi 的追番列表

== Description ==

在文章或页面内使用短代码 `[bangumi]` 展示追番列表 [Demo](https://yuncaioo.com/post-1135.html)

本插件使用 [Bangumi API](http://bgm.tv)，API 地址为 `https://api.bgm.tv`

[Github: Bangumi-List](https://github.com/fhyuncai/Bangumi-List)

== Installation ==

1. 在 WordPress 插件搜索栏搜索插件，或前往 GitHub 下载最新版本的插件上传，安装并启用
2. 前往 `设置 > Bangumi 追番列表` 配置相关必要信息
3. 在文章或页面内使用短代码 [bangumi]
